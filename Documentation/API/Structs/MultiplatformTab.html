MultiplatformTab

public struct MultiplatformTab : View

Holds the data for a tab that can be added to the MultiplatformTabBar control.

Example:

.tab(title: "Categories", icon: Image(systemName: "tray.fill")) {
    VStack {
       Text("Body of Contents")
    }
}
